## Introduction
This project was developed as part of the SC2006, Software Engineering, module assignment at Nanyang Technological University, Singapore.

**Eventure** is a web application, built for both desktop and mobile, that **simplifies event discovery in Singapore** through an **interactive, swipe-based interface**.

The platform aggregates event listings from sources like _Ticketmaster_ while **empowering small local organisers to create and promote their own events**. Built with software engineering best practices in mind, _**Eventure**_ supports Singapore's Smart Nation initiative by enhancing community engagement and enabling digital transformation for local SMEs.
